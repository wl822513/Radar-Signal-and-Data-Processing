# 雷达信号处理和数据处理

本项目用于记录雷达信号处理的学习与实验过程，包括信号建模、滤波、目标检测与跟踪等模块。

## 📘 知识点

1. 包络（Envelope）：信号幅度随时间变化的那条曲线。
   - 直观理解：包络就是信号“外形的轮廓线”，描述信号强弱随时间的变化。
   - 数学表达：对于调制信号
     s(t) = A(t) * cos(2π f_c t + φ(t))
     其中 A(t) 就是信号的包络，φ(t) 是瞬时相位。
   - 复包络形式：为了便于数学分析和信号处理，常用复数表示
     s(t) = A(t) * exp(j φ(t))
     这里的复包络 s(t) 提取了信号幅度和相位变化，而不直接考虑高频载波。
   - 工程意义：
     - 在雷达中，包络可以表示目标回波强度随时间的变化；
     - 匹配滤波或解调后，包络峰值对应目标存在的时刻；
     - 分析包络可以判断信号能量分布、脉冲形状和目标回波特性。
